This repository contains the complete materials for the companion paper deriving infrared Yang-Mills confinement from the KPP (Kolmogorov-Petrovsky-Piskunov) flow.

The paper derives an effective scalar-tensor field theory of IR confinement directly from SU(N) Yang-Mills via a Hubbard-Stratonovich transformation on the composite operator Ω = g²FF/16π². The spectral density satisfies a coupled KPP flow equation whose fixed-point structure encodes the entire IR physics.
